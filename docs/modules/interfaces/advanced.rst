Advanced interfaces
-------------------

More customization can be realized using advanced module interfaces. These are...



.. list-table:: Advanced interfaces
   :widths: 30 70
   :header-rows: 1

   * - Interface
     - About
   * - :ref:`propertyinterface`
     - Used to define custom property editors in the user interface.
   * - :ref:`collectioninterface`
     - Used to define collections which then are shown in GTlab's collection browser.
   * - :ref:`networkinterface`
     - Used to define a module-related network connection.
   * - :ref:`calcexecinterface`
     - Used to define separate options to execute calculators which not in the basic execution system.
